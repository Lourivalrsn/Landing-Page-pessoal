Projetos

Projeto 1

Desenvolvi um jogo de nave 2D em Java, focado em proporcionar uma experiência dinâmica e divertida. Nesse projeto, o jogador controla uma nave espacial, enfrentando ondas de inimigos em um ambiente espacial. Usei conceitos de orientação a objetos para estruturar o código, organizando as classes em elementos como a nave, inimigos, projéteis e sistema de pontuação.

Projeto 2

Esse projeto foi criado com o objetivo de desenvolver uma aplicação web prática e intuitiva para gerenciamento de compromissos. Junto com um grupo de quatro amigos, projetamos e implementamos uma interface funcional utilizando HTML para a estrutura, CSS para a estilização e JavaScript para a interatividade. O sistema permite aos usuários criar, editar e excluir eventos, além de visualizar compromissos futuros de forma organizada.

Projeto 3

Desenvolvi um mini jogo de cartas em C, focado em criar uma experiência interativa e divertida. O jogo inclui funcionalidades como embaralhar, distribuir cartas e aplicar regras básicas para simular uma partida de cartas.
